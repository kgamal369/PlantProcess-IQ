# PlantProcess IQ — One-Page Pilot Offer

## Pilot name
Process-to-Quality Risk Diagnostic Pilot

## Customer problem
Manufacturing plants collect large amounts of process and quality data, but the data is fragmented across MES, L2, historian, lab, inspection, downtime, and ERP systems. This makes defect investigation slow and makes early quality-risk prediction difficult.

## Pilot promise
PlantProcess IQ connects historical process and quality exports, builds material/batch genealogy, checks data quality, identifies process-to-defect correlations, generates material risk scores, and provides investigation reports for process and quality teams.

## Pilot scope
- One site
- One production route
- One or two defect groups
- CSV/Excel/SQL export input
- Read-only analysis
- Synthetic/anonymized examples for demo
- Dashboard summary and PDF investigation report

## Deliverables
- Data readiness assessment
- Canonical data mapping
- Data-quality issue report
- Correlation diagnostic summary
- Initial rule-based risk scoring
- Investigation report sample
- Go/no-go roadmap for ML pilot

## Important positioning
The pilot provides correlation and risk-support evidence. It does not claim guaranteed root cause without process-engineer validation.
